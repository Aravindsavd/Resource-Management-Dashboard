const API = 'http://localhost:3000/api';

  unattachedNICs:  { title:'Unattached Network Interfaces', sub:'NICs with no VM attached — may be holding Public IPs.', label:'Unattached NICs', navId:'nav-nics' },
  unusedNSGs:      { title:'Unused Network Security Groups',sub:'NSGs not associated with any subnet or NIC.', label:'Unused NSGs', navId:'nav-nsgs' },
  emptyLBs:        { title:'Empty Load Balancers',          sub:'Load Balancers with no backend pool members.', label:'Empty Load Balancers', navId:'nav-lbs' },
};

const RES_TAB_COLORS = { stoppedVMs:'#2563eb', unattachedDisks:'#d97706', unattachedPIPs:'#dc2626', unattachedNICs:'#7c3aed', unusedNSGs:'#0891b2', emptyLBs:'#65a30d' };

let state = { data: null, currentPage: 'stoppedVMs', tab: 'all', page: 1, filtered: [] };

function toast(msg, dur=3000) {
  const t=document.getElementById('toast');
  document.getElementById('toast-msg').textContent=msg;
  t.classList.remove('hide');
  clearTimeout(t._t);
  t._t=setTimeout(()=>t.classList.add('hide'),dur);
}

async function runScan(force=false) {
  const btn=document.getElementById('scan-btn');
  btn.disabled=true;
  btn.innerHTML='<i class="ti ti-refresh spin"></i>Scanning…';
  document.getElementById('scan-title').textContent='Scanning Azure…';
  document.getElementById('scan-sub').textContent='This may take 1–3 minutes across all subscriptions. Please wait.';
  setTableLoading();
  toast('Scanning Azure… this may take a few minutes.', 180000);

  // Start scan with a long timeout (5 min)
  const controller = new AbortController();
  const timeout = setTimeout(() => controller.abort(), 300000);

  try {
    const res = await fetch(`${API}/scan${force?'?force=true':''}`, { signal: controller.signal });
    clearTimeout(timeout);
    if(!res.ok) {
      const errText = await res.text();
      throw new Error(errText);
    }
    const data = await res.json();
    if(data.error) throw new Error(data.error);
    state.data = data;
    updateUI();
    toast(data.fromCache ? 'Loaded from cache' : `Scan complete — ${data.summary?.totalUnused ?? 0} unused resources found!`, 5000);
  } catch(e) {
    clearTimeout(timeout);
    if(e.name === 'AbortError') {
      document.getElementById('scan-title').textContent='Scan timed out';
      document.getElementById('scan-sub').textContent='Azure scan took too long. Try scoping to one subscription.';
      toast('Scan timed out after 5 minutes', 5000);
    } else {
      document.getElementById('scan-title').textContent='Connection failed';
      document.getElementById('scan-sub').textContent='Make sure the backend is running: node server.js';
      toast('Error: ' + e.message.slice(0,100), 6000);
    }
  } finally {
    btn.disabled=false;
    btn.innerHTML='<i class="ti ti-radar"></i>Run Scan Now';
  }
}

function updateUI() {
  const s = state.data?.summary;
  if(!s) return;
  document.getElementById('s1-val').textContent = s.totalStoppedVMs ?? '—';
  document.getElementById('s2-val').textContent = s.billedVMs ?? '—';
  document.getElementById('s3-val').textContent = s.totalMonthlyWaste != null ? '$'+s.totalMonthlyWaste.toLocaleString() : '—';
  document.getElementById('s4-val').textContent = s.totalUnused ?? '—';

  const ts = s.scannedAt ? new Date(s.scannedAt) : null;
  document.getElementById('scan-title').textContent = ts ? 'Last scan: '+ts.toLocaleDateString()+' at '+ts.toLocaleTimeString() : 'Scan complete';
  document.getElementById('scan-sub').textContent = `${s.totalUnused} unused resources found across all subscriptions`;

  document.getElementById('nb-vms').textContent   = s.totalStoppedVMs ?? 0;
  document.getElementById('nb-disks').textContent = s.unattachedDisks ?? 0;
  document.getElementById('nb-pips').textContent  = s.unattachedPIPs ?? 0;
  document.getElementById('nb-nics').textContent  = s.unattachedNICs ?? 0;
  document.getElementById('nb-nsgs').textContent  = s.unusedNSGs ?? 0;
  document.getElementById('nb-lbs').textContent   = s.emptyLBs ?? 0;

  buildResTabs();
  renderCurrentPage();
}

function buildResTabs() {
  const s = state.data?.summary || {};
  const counts = { stoppedVMs: s.totalStoppedVMs, unattachedDisks: s.unattachedDisks, unattachedPIPs: s.unattachedPIPs, unattachedNICs: s.unattachedNICs, unusedNSGs: s.unusedNSGs, emptyLBs: s.emptyLBs };
  document.getElementById('res-tabs').innerHTML = Object.entries(PAGES).map(([k,v])=>`
    <div class="rtab${state.currentPage===k?' active':''}" onclick="setPage('${k}')">
      <span class="dot" style="background:${RES_TAB_COLORS[k]}"></span>${v.label}
      <span style="font-size:11px;font-weight:600;margin-left:2px">(${counts[k]??0})</span>
    </div>`).join('');
}

function setPage(pg) {
  if(pg==='dashboard') return;
  state.currentPage=pg; state.tab='all'; state.page=1;
  document.querySelectorAll('.nav').forEach(n=>n.classList.remove('active'));
  const navEl=document.getElementById(PAGES[pg].navId);
  if(navEl) navEl.classList.add('active');
  document.getElementById('pg-title').innerHTML=PAGES[pg].title;
  document.getElementById('pg-sub').textContent=PAGES[pg].sub;
  document.getElementById('page-label').innerHTML=PAGES[pg].label;
  document.getElementById('search').value='';
  document.querySelectorAll('.tab').forEach(t=>t.classList.remove('active'));
  document.querySelector('.tab').classList.add('active');
  buildResTabs();
  renderCurrentPage();
}

function setTab(tab, el) {
  state.tab=tab; state.page=1;
  document.querySelectorAll('.tab').forEach(t=>t.classList.remove('active'));
  el.classList.add('active');
  applyFilters();
}

function applyFilters() {
  const raw = state.data?.resources?.[state.currentPage] || [];
  const q = document.getElementById('search').value.toLowerCase();
  state.filtered = raw.filter(r => {
    const matchTab =
      state.tab==='all' ||
      (state.tab==='deallocated' && r.powerState && r.powerState.toLowerCase().includes('deallocated')) ||
      (state.tab==='billed'      && r.stillBilled);
    const matchQ = !q ||
      (r.name||'').toLowerCase().includes(q) ||
      (r.resourceGroup||'').toLowerCase().includes(q) ||
      (r.subscription||'').toLowerCase().includes(q) ||
      (r.location||'').toLowerCase().includes(q);
    return matchTab && matchQ;
  });
  state.page=1;
  renderTable();
}

function renderCurrentPage() {
  if(!state.data) return setTableLoading();
  const raw = state.data.resources?.[state.currentPage] || [];
  const billed = raw.filter(r=>r.stillBilled).length;
  const deal   = raw.filter(r=>r.powerState && r.powerState.toLowerCase().includes('deallocated')).length;
  document.getElementById('tc-all').textContent  = raw.length;
  document.getElementById('tc-deal').textContent = deal;
  document.getElementById('tc-bill').textContent = billed;
  const showVmTabs = state.currentPage==='stoppedVMs';
  document.getElementById('tab-deal').style.display = showVmTabs ? '' : 'none';
  document.getElementById('tab-bill').style.display = showVmTabs ? '' : 'none';
  state.filtered = raw;
  renderTable();
}

const HEADS = {
  stoppedVMs:      [['chk','3%'],['Subscription','14%'],['VM Details','22%'],['Location & Size','14%'],['State & Type','14%'],['Stopped Duration','11%'],['Billed/mo','10%'],['Actions','8%']],
  unattachedDisks: [['chk','3%'],['Subscription','16%'],['Disk Name','22%'],['Resource Group','18%'],['Location','11%'],['Size & SKU','14%'],['Est. Cost/mo','10%'],['Actions','8%']],
  unattachedPIPs:  [['chk','3%'],['Subscription','16%'],['IP Name','22%'],['Resource Group','18%'],['Location','11%'],['SKU','14%'],['Est. Cost/mo','10%'],['Actions','8%']],
  default:         [['chk','3%'],['Subscription','16%'],['Name','24%'],['Resource Group','20%'],['Location','13%'],['Est. Cost/mo','12%'],['Actions','8%']],
};

function renderTable() {
  const heads = HEADS[state.currentPage] || HEADS.default;
  document.getElementById('tbl-head').innerHTML = '<tr>'+heads.map(([h,w])=>`<th style="width:${w}">${h==='chk'?'<input type="checkbox" id="chk-all" onchange="toggleAll(this)">':h}</th>`).join('')+'</tr>';

  const slice = state.filtered.slice((state.page-1)*PAGE_SIZE, state.page*PAGE_SIZE);
  if(slice.length===0) {
    document.getElementById('tbl-body').innerHTML = `<tr class="state-row"><td colspan="${heads.length}"><i class="ti ti-server-off"></i>No resources match your filter.</td></tr>`;
  } else {
    document.getElementById('tbl-body').innerHTML = slice.map(r=>renderRow(r)).join('');
  }
  const total=state.filtered.length, start=(state.page-1)*PAGE_SIZE+1, end=Math.min(state.page*PAGE_SIZE,total);
  document.getElementById('pag-info').textContent = total ? `Showing ${start}–${end} of ${total} results` : 'No results';
  renderPag();
}

function renderRow(r) {
  const pg = state.currentPage;
  const det = `<td><button class="btn-det" onclick='showDetail(${JSON.stringify(JSON.stringify(r))})'>Details</button></td>`;
  const chk = `<td><input type="checkbox" class="row-chk"></td>`;
  const sub = `<td><div class="sub-name">${r.subscription||'—'}</div><div class="sub-id">${(r.subscriptionId||'').substring(0,8)}…</div></td>`;
  const cost = `<td><div class="cost-v">$${(r.estMonthlyCost||0).toFixed(2)}</div><div class="cost-n">Est./mo</div></td>`;

  if(pg==='stoppedVMs') {
    const ps=r.powerState||'';
    const badge=r.stillBilled?'<span class="badge bs">Stopped</span>':'<span class="badge bd">Deallocated</span>';
    const billing=r.stillBilled?'<div class="billed-warn">⚠ Compute billed</div>':'<div class="free-note">Compute FREE</div>';
    const days=r.stoppedDays||0;
    const durCls=days>90?'hi':days>30?'md':'';
    const durTxt=days>90?'> 90 days':days>30?'> 30 days':days+' days';
    return `<tr>${chk}${sub}
      <td><div class="vm-name"><i class="ti ti-server" style="font-size:14px"></i>${r.name}</div><div class="vm-rg">⊙ ${r.resourceGroup}</div></td>
      <td><div>${r.location}</div><span class="size-tag">${r.size||''}</span></td>
      <td>${badge}${billing}</td>
      <td><div class="dur ${durCls}"><i class="ti ti-clock"></i>${durTxt}</div></td>
      <td><div class="cost-v">$${(r.diskCost||r.estMonthlyCost||0).toFixed(2)}</div><div class="cost-n">Disks + IPs</div></td>
      ${det}</tr>`;
  }
  if(pg==='unattachedDisks') {
    return `<tr>${chk}${sub}
      <td><div class="vm-name"><i class="ti ti-device-floppy" style="font-size:14px"></i>${r.name}</div></td>
      <td><div class="vm-rg">${r.resourceGroup}</div></td>
      <td>${r.location}</td>
      <td><div>${r.sizeGB} GB</div><span class="size-tag">${r.sku||''}</span></td>
      ${cost}${det}</tr>`;
  }
  if(pg==='unattachedPIPs') {
    return `<tr>${chk}${sub}
      <td><div class="vm-name"><i class="ti ti-world" style="font-size:14px"></i>${r.name}</div></td>
      <td><div class="vm-rg">${r.resourceGroup}</div></td>
      <td>${r.location}</td>
      <td><span class="size-tag">${r.sku||'Basic'}</span></td>
      ${cost}${det}</tr>`;
  }
  return `<tr>${chk}${sub}
    <td><div class="vm-name">${r.name}</div><div class="vm-rg">${r.resourceGroup}</div></td>
    <td>${r.location}</td>
    ${cost}${det}</tr>`;
}

function setTableLoading() {
  document.getElementById('tbl-head').innerHTML='';
  document.getElementById('tbl-body').innerHTML=`<tr class="state-row"><td colspan="8"><i class="ti ti-refresh spin"></i>Loading data from Azure…</td></tr>`;
}

function renderPag() {
  const total=Math.ceil(state.filtered.length/PAGE_SIZE);
  let html=`<div class="pb" onclick="goPage(${state.page-1})"><i class="ti ti-chevron-left" style="font-size:13px"></i></div>`;
  for(let i=1;i<=Math.min(total,7);i++) html+=`<div class="pb${i===state.page?' act':''}" onclick="goPage(${i})">${i}</div>`;
  if(total>7) html+=`<div class="pb" style="border:none;cursor:default">…</div>`;
  html+=`<div class="pb" onclick="goPage(${state.page+1})"><i class="ti ti-chevron-right" style="font-size:13px"></i></div>`;
  document.getElementById('pag-btns').innerHTML=html;
}

function goPage(p) {
  const max=Math.ceil(state.filtered.length/PAGE_SIZE);
  if(p<1||p>max) return;
  state.page=p; renderTable();
}

function toggleAll(el) { document.querySelectorAll('.row-chk').forEach(c=>c.checked=el.checked); }

function showDetail(jsonStr) {
  const r=JSON.parse(jsonStr);
  document.getElementById('modal-title').textContent=r.name;
  const rows=[
    ['Resource type', state.currentPage.replace(/([A-Z])/g,' $1').trim()],
    ['Subscription', r.subscription||'—'],
    ['Subscription ID', r.subscriptionId||'—'],
    ['Resource group', r.resourceGroup||'—'],
    ['Location', r.location||'—'],
    r.size ? ['VM size', r.size] : null,
    r.powerState ? ['Power state', r.powerState] : null,
    r.sizeGB ? ['Disk size', r.sizeGB+' GB'] : null,
    r.sku ? ['SKU', r.sku] : null,
    r.stillBilled!=null ? ['Still billed', r.stillBilled ? '⚠ Yes — compute charges active' : 'No — deallocated'] : null,
    ['Est. monthly cost', '$'+(r.estMonthlyCost||r.diskCost||0).toFixed(2)],
  ].filter(Boolean);
  const tags=r.tags&&Object.keys(r.tags).length ? `<div class="detail-row"><div class="detail-key">Tags</div><div class="detail-val"><div class="tags-wrap">${Object.entries(r.tags).map(([k,v])=>`<span class="tag-pill">${k}: ${v}</span>`).join('')}</div></div></div>` : '';
  document.getElementById('modal-body').innerHTML=rows.map(([k,v])=>`<div class="detail-row"><div class="detail-key">${k}</div><div class="detail-val ${k.includes('ID')?'detail-mono':''}">${v}</div></div>`).join('')+tags;
  document.getElementById('overlay').classList.add('show');
}

function closeModal() { document.getElementById('overlay').classList.remove('show'); }

function exportCSV() {
  const data=state.filtered;
  if(!data.length) return toast('Nothing to export');
  const keys=Object.keys(data[0]).filter(k=>k!=='tags');
  const rows=[keys,...data.map(r=>keys.map(k=>JSON.stringify(r[k]??'')))];
  const csv=rows.map(r=>r.join(',')).join('\n');
  const a=document.createElement('a');
  a.href='data:text/csv;charset=utf-8,'+encodeURIComponent(csv);
  a.download=`cloudops-${state.currentPage}-${new Date().toISOString().slice(0,10)}.csv`;
  a.click();
  toast('CSV downloaded');
}
